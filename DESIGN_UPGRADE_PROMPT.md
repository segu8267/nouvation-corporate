# NOUVATION コーポレートサイト - デザイン改善プロンプト

## 📋 現在のサイト状況

### 技術スタック

- **フレームワーク**: Astro v4.0.0（静的サイト生成）
- **スタイリング**: Tailwind CSS v3.4.0
- **言語**: TypeScript 5.9.3
- **CMS**: MicroCMS（ニュース記事管理）
- **デプロイ**: Vercel
- **URL**: https://nouvation-corporate.vercel.app/

### サイト構成（1ページ構成）

```
Header（ヘッダー）
  ↓
FV（ファーストビュー）
  ↓
News（お知らせ）- MicroCMS連携
  ↓
Social Issues（社会課題）
  ↓
Mission（ミッション）
  ↓
Services（4つのサービス紹介）
  ├─ 01. 市場価格推移グラフ（メイン）
  ├─ 02. Instagramメディア
  ├─ 03. 農業AI専用サポート（準備中）
  └─ 04. 農業特化型Web制作
  ↓
Story（創業者ストーリー）
  ↓
Voice（顧客の声）
  ↓
FAQ（よくある質問）
  ↓
Footer（フッター）
```

### 現在のデザイン要素

#### カラーパレット

- **メインカラー**: `#d5db00` (黄緑・ライム)
- **ダークグレー**: `#464237`, `#352421`, `#231815`
- **ベージュ系**: `#F6F6EE`, `#F0ECDE`
- **ダークベース**: `#111111`
- **テキスト**: `#E5E7EB`（白系）

#### フォント

- **本文**: Noto Sans JP
- **見出し**: Zen Maru Gothic
- **英語**: YakuHanJP

#### 既存のアニメーション

- **Fade Up**: スクロールで要素がフェードインしながら上昇
- **Marquee**: フッターでロゴが流れる
- **Smart Header**: スクロール時に表示/非表示が切り替わる

#### デザイン特徴

- 余白を広めに取った、ゆったりとしたレイアウト
- ノイズテクスチャで質感を追加
- グリッドパターン（一部セクション）
- 背景色に応じてヘッダーの色が変化（白⇔黒）

## 🔧 技術的な制約

### 使用可能な技術

- ✅ Tailwind CSS（既存のクラスを活用）
- ✅ Vanilla JavaScript（Astroの`<script>`タグ内）
- ✅ CSS アニメーション・トランジション
- ✅ Intersection Observer API（すでに使用中）
- ✅ GSAP、Anime.js などの軽量ライブラリ（必要であれば）

### 避けたい技術

- ❌ React、Vue などのフレームワーク（Astroの静的サイト生成を維持）
- ❌ 重すぎるライブラリ（パフォーマンス重視）
- ❌ Three.js などの3D系（今回は不要）

### パフォーマンス要件

- Lighthouse スコア 90点以上を維持
- 初回表示は高速に
- モバイルでもスムーズに動作

## 📌 補足情報

- 開発環境: macOS、VSCode

- 既存のアニメーションは `Intersection Observer` で実装済み
