# NOUVATION Corporate Site

農業界に情報で革新を。NOUVATION コーポレートサイト

## 🚀 技術スタック

- **Framework**: Astro v4
- **Styling**: Tailwind CSS
- **CMS**: MicroCMS
- **Deploy**: Vercel

## 🛠️ 開発環境セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# プレビュー
npm run preview
```

## 🌐 環境変数

`.env.example`をコピーして`.env`を作成し、必要な値を設定してください。

```bash
cp .env.example .env
```

## 📦 デプロイ

Vercel に自動デプロイされます。

- **本番**: https://nouvation.com
- **プレビュー**: Pull Request 作成時に自動生成
